####ROM特性

- 完美root权限;
- 精简部分无用软件;
- 精简多余lib库文件;
- 精简自带壁纸;
- ROM体积精简到130M;
- 仿HTC美化;
- 更换开机动画为Google动画;
- 抽屉背景透明，显示壁纸;
- 网络，网速优化;
- 默认开启未知来源，USB调试等;
- 默认关闭自动旋转，GPS，拨号盘触摸音效，触摸提示音，锁屏提示音等;
- 默认最低亮度;
- 去除了谷歌框架，空闲更多的可用内存;
- 省电增强;
- 去除照相声音与开机铃声;
- 替换自带输入法为百度输入法;
- 加入原生归属地;
- 加入RE管理器;
- 桌面、抽屉重新排版;
- 内置实用软件： 释放内存 百度输入法;
- 其他优化请大家刷入后体验!
 
####再精簡

- 精簡百度輸入法;
- 精簡部分無用的內置APP

####注意事項

- 此ROM为卡刷包，下载后放入SD卡根目录，进入recovery，按照刷机教程即可刷入！
- 刷前勿必双清, 可能的话三清
